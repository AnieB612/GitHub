### CONFIGURAÇÃO BÁSICA
___________________

* Versão do Programa Instalado
  git --version

* Comando para ver as informações a respeito do GIT
  git config

* Configurar o nome
  git config --global user.name "anie"
  
* Configurar o email
  git config --global user.email y_anie@hotmail.com
